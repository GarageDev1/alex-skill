# 去 AI 味案例库

这是 `SKILL.md` 的配套例子。写完一段拿不准的话，回来对照着查：左边是改之前的 AI 味写法，右边是改之后落地的写法，最后一列说清楚毛病出在哪。

例子源自财经研报的真实改稿，但毛病和改法对任何中文文章都通用——架子句、贴标签、情绪比喻、否定对仗、堆砌对冲，换个题材一样要改。

## 一、整句改写对照

| 改前（AI 味） | 改后（落地） | 毛病 |
|---|---|---|
| 四个看多支柱里三个事后大体兑现，买在 2021 年顶点却至今仍亏约一半——决定回报的是出价贵不贵，而非基本面对不对 | （整句删掉，直接进结论和分点） | 只搭架子、没给信息、破折号、否定对仗 |
| 估值消化是唯一致命变量 | 估值的下跌幅度超过了盈利的增长幅度 | 贴标签代替讲机制 |
| 利润增长的"确定性"在 2025 年彻底破产 | 原本被认为确定的利润增长，2025 年全面转为下滑 | 情绪词"破产" |
| 茅台上市首次双降、洋河净利 −67%、五粮液会计暴雷、泸州老窖 −20% | 茅台上市以来营收净利首次双降，洋河净利润降 67%，泸州老窖降 20%，五粮液因会计差错更正大幅下滑 | 黑话"暴雷"、电报体堆叠 |
| 人均消费从未"升级"，只是高端化撑了几年 | 人均饮酒量一直在下降，所谓升级只是高端酒提价支撑了几年收入 | 太模糊，没说清"升级"指什么 |

## 二、词和短语替换总表

### 情绪、暴力、比喻词 → 换成字面

| 改前 | 改后 |
|---|---|
| 估值消化 / 杀估值 | 估值下跌 |
| 单边杀估值吃掉利润增长 | 估值下跌幅度超过利润增长 |
| 会计暴雷 | 因会计差错更正大幅下滑 |
| 也崩了 / 崩盘 | 开始下跌 / 大幅下跌 |
| 近乎清零 | 降 67%（用实际数） |
| 彻底破产（指假设失效） | 全面转为下滑 / 不成立 |
| 蛋糕在缩 | 行业总量在萎缩 |
| 份额赢家 | 份额上升的公司 |
| 护城河救不了股东 | 集中度提升没有给股东带来回报 |
| 最后一根柱子也塌了 | 收入和利润也开始下降 |
| 价盘松动 | 批价下跌 |
| 收割期 / 进入收割 | 盈利期 / 进入盈利 |
| 打到同一条能力线上 | 达到几乎相同的能力水平 |
| 贴身差距 | 只差 1 分（用实际数） |

### 设问、开场垫话 → 删掉，直接给结论

| 改前 | 改后 |
|---|---|
| 那 X 什么时候会 Y？（设问自答开头） | 删设问，直接给结论 |
| 答案很清楚 / 答案是 | 删，直接陈述 |
| 问题来了 / 值得一问的是 | 删 |
| 说到底 / 归根结底 / 本质上 / 说白了 | 删，直接说实质 |
| X 把这件事钉死了 / 一锤定音 / 板上钉钉（开场甩结论） | 删，直接摆事实 |

### 反转金句、比喻二选一 → 落到字面

| 改前 | 改后 |
|---|---|
| 这不是 X，是 Y（反转金句） | 直接讲 Y 是什么 + 机制（"这不是亏损，是预付款"→"计入当期成本，对应未来交付的算力产能"） |
| 是 A 还是 B（A、B 都是比喻） | 写两种字面情况 + 怎么区分（"是地基在塌还是工地在挖坑"→"结构性恶化还是阶段性加大投入"） |
| 不是 X，而是 Y（X 是稻草人） | 删"不是 X"，直接陈述 Y |

### 口号、拔高 → 删掉，或落到事实

| 改前 | 改后 |
|---|---|
| 最该记住的一条 | 删 |
| 最反直觉的结论 | 结论是 |
| 本报告核心一图 | 下图对比两者走势 |
| 这轮最贵的一课 / 给信仰者上的一课 | 删 |
| 起点估值是最不可逆的那个变量 | 买入时的估值对长期回报影响最大，事后很难弥补 |
| 强者恒强 | 继续上升 |
| 讽刺的是 | 删 |
| 兑现得最干净 | 兑现得最确定 |
| 这是当前最硬的证据 / 这已经说明一切（结尾强行拔高） | 删，让前面的事实自己说话 |

### 否定排比、对仗 → 拉平

| 改前 | 改后 |
|---|---|
| 决定回报的是出价贵不贵，而非基本面对不对 | 决定盈亏的是买入时的估值高低，不是行业判断准不准 |
| 基本面向上、估值向下、股价净跌（三段对仗） | 利润上升、估值下降，两者相抵后股价净跌 |
| 集中度提升 ≠ 股东赚钱 | 集中度提升并没有让股东赚钱 |

### 伪框架、标签式概括 → 删掉或填实

| 改前 | 改后 |
|---|---|
| 四个看多支柱里三个大体兑现……（只搭架子） | 删掉，改成直接列四个问题的结论 |
| 分化为三派：死守派 / 反思派 / 反复派 | 直接写每个人怎么做（刘彦春满仓、焦巍清仓后回补、张坤萧楠反复） |

### 程度词 → 换成数字

| 改前 | 改后 |
|---|---|
| 大幅下滑 | 下降 67% |
| 显著提升 | 从 65% 升到 78% |
| 大体兑现 | 基本兑现（或具体说兑现了哪几项） |

## 三、对冲词清单

有两种"对冲"，一种该删，一种该留，分清楚。

**堆砌型对冲词，一句最多留一个，多了就删。** 这类是对观点心虚：可能、或许、也许、大概、似乎、貌似、在某种程度上、一定程度上、某种意义上、被认为、据称、据说、相对而言、总体而言、整体来看。

想表达不确定，用一个明确的词就够，别叠起来写成"该政策或许在某种程度上可能被认为产生某些影响"。

**数据估算时的对冲该留，那是标口径，不是含糊。** 约、左右、近、超过、不足、~ 这些是诚实地标注估算精度。"约 7366 亿""~20 倍""近 74%""基本持平"都属于这一类，照写。

区别就在这：对冲观点是心虚，对冲数据精度是严谨。
