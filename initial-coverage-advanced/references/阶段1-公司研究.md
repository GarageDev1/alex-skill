# 阶段 1：公司研究

产物：`[公司]_公司研究_[日期].md`，6000–9000 字。这是后面四个阶段的素材库，也是研报正文一、二、三章的底稿。

公司研究不是把资料分门别类地罗列，而是按投资逻辑链组织——读完这一稿，应该能回答三个问题：这是一家什么公司、它所在的赛道有多大、它凭什么占据这块市场空间。每一节都为后面的投资论点服务。

开始前先确认标的所在市场：在 A 股、港股还是美股，决定了披露平台（巨潮资讯网 / 港交所披露易 / SEC EDGAR）、会计准则和币种。

开始同行与竞争研究前，同时读 `同业与产业链横向比较.md`。阶段 1 要完成 O/V/T 三层标的池的候选建池、取数和第一版横向结论，供阶段 3–5 直接复用。

## 取数

四类数据，每类对应独立检索：

| 类别 | 主来源 |
|------|--------|
| 公司基本面 | 年报/中报/季报、招股书、公司公告、公司 IR 页 |
| 业务与经营 | 财报的分部数据、业绩说明会纪要、公司官网产品页 |
| 行业与竞争 | 行业研究机构（IDC、Counterpoint、艾瑞、灼识）、协会数据、竞品财报 |
| 市场数据 | 股价、市值、股本、一致预期（当次检索，带日期） |
| 横向比较 | 同一行情日的市值/EV + 最新 LTM/FY1 收入、归母净利润、PE、PS、关键经营指标；先写 staging，校验后合并 |

信任分级和双源印证按 SKILL.md 的取数标准执行。

**建数据底稿**（和阶段2 同一套五字段）：开一份 `[公司]_公司研究_data` 底稿，每个进稿的数字先写进去——`数值 | 口径 | 时间点 | 来源（含链接）| 置信度（高/中/低 + 单源/双源）`。**底稿里没有的数，不许写进稿子**，正文数字和后面要画进图的数据都算。取不到的标「未找到」，写明用什么假设替代、依据是什么。口径就近标在数字旁（招股书截止日 / 年报、绿鞋前后、账面 / 经调整），全稿统一。关键数据（基年分部收入、行业规模、市占率）要双源，且双源必须异源——别拿转引同一篇的两篇当双源。

## 写什么

### 点题设问（供摘要开头，不单独成引言段）

写一句点题设问，抛出这家公司最值得研究的那个问题。看标杆：中泰-腾讯用"为何腾讯的产品总能后来居上""为何日赚 5 亿仍能大象起舞"这种设问当钩子。这句是切入点，不是公司简介。

注意：报告**不写独立的"引言"段**。这句设问移到报告摘要的开头当钩子（见 `assets/报告模板与版式.md`），阶段 5 组装时正文直接从"一、公司概况"起。这里产出这句，是为摘要开头备料。

### 一、公司概况

**历史复盘**。沿革按阶段分（不是流水账），每个阶段点出战略转折和它对今天的意义。配一张历史沿革图（时间轴 + 关键节点 + 收入规模）。上市公司再做一段**股价复盘**——把历史股价和重大事件、政策、业绩拐点对齐，看清股价对什么敏感。这段为后面"为什么是现在"做铺垫。

**业务布局**。核心动作是**分部拆分**：把营收按业务/产品线拆开，每个分部的收入、占比、增速、变现模式讲清。配分部收入图、分部毛利率图。讲清楚公司的盈利来源、各块业务的财务特征有什么不同——这直接决定阶段 2 的模型按单线还是分部建。

**股权与治理**。股权结构、实控人、一致行动人、重要股东背景（有没有国资/产业资本）、组织架构。配股权结构图。

**管理层**。核心高管 3–5 人，每人讲背景、履历、任职年限、关键成就。技术驱动的公司重点讲核心技术团队的学术和产业背景。

**财务分析**。历史收入、利润、毛利率、费用率、现金流、ROE 的趋势，判断公司处在什么阶段（高增长 / 成熟 / 拐点）。点出财务上的关键特征，比如腾讯的"EPS 增速 > 净利润增速 > 毛利润增速 > 收入增速"高质量增长范式，或寒武纪的"营收阶梯式增长 + 盈利端扭亏转盈"拐点。

### 二、行业空间

把公司放到行业背景中考察。市场规模（带来源、年份、预测 CAGR）、增长驱动、产业链位置、政策环境。多业务公司按分部分别讲空间（腾讯拆成 2C 的"现金牛"和 2B 的"魔法盒"）。

行业规模数要权威来源（IDC、灼识、Wind、协会），标清是历史值还是预测值。配市场规模图、产业链图。

### 三、竞争格局

谁在与它竞争、它处在什么位置。按 `同业与产业链横向比较.md` 建三个允许重叠、用途不同的池，不把“直接竞争”“估值可比”和“同属概念”混成一件事：

1. **经营竞争对标池（O 池）**：纳入直接竞品、产品替代者和分部层面对手，用于比较市占率、产品、客户、产能、增长和利润率。综合集团优先取可比分部数据；拿不到时标“集团口径污染”。
2. **严格估值可比公司池（V 池）**：按核心业务、商业模式、收入与利润驱动、周期和资本强度筛选。目标 5–10 家；纯粹可比稀缺时保留真实的 3–4 家并说明局限，不用明显不可比的公司凑数。
3. **核心概念股 / 产业链主题池（T 池）**：纳入上下游、互补环节和平台型龙头，标清每家所处环节及与目标公司的关系。它回答资本市场如何给同一产业链不同利润池定价，不直接用于目标倍数。

三层都把目标公司放在首行或高亮行。先在独立 staging Excel/CSV 增量取数并校验，阶段 1 研究稿只写建池理由和第一版结论；以下三种完整视图在阶段 3 合并进正式 Excel，不在 Markdown 手抄一套最终表：

- **O 池经营对标表**：至少含直接竞争关系、收入、利润、收入增速、毛利率、净利率、市占率及行业关键经营指标，并标是否进入 V 池。
- **V 池核心数据与估值表**：至少含代码/市场、可比理由、市值、收入 LTM、归母净利润 LTM、收入增速、净利率、PE TTM/FY1、PS TTM/FY1、行业关键经营指标；另给有效样本分位统计。
- **T 池产业链位置表**：至少含产业链环节、与目标公司的关系、市值、收入、利润、增长、净利率、PE FY1、PS FY1 和可验证的主题暴露指标。

历史财务统一 LTM、预测倍数统一 FY1 或 NTM，市值使用同一行情日并统一展示币种。负利润 PE 写 `N/M`，不纳入 PE 分位；双重上市去重；综合集团优先用分部数据。配市占率图、经营对标表和产业链定价光谱图。

表后写 3–5 条判断：目标公司的经营指标在 O 池处于什么位置，规模与盈利在 V 池处于什么位置，估值溢价/折价能否被增长、利润率、壁垒或周期解释，T 池把最高定价给了哪个产业链环节，以及哪个经营变量会改变目标公司的组别。点出差异化优势——这是后面“竞争壁垒”和阶段 3 估值的共同素材。

### 风险

8–12 条，分类列（宏观/行业、经营、政策监管、技术/竞争、财务/估值）。每条写清触发条件和影响，不要笼统。

## 自检

- 三个核心问题（什么公司 / 多大空间 / 凭什么）都答上了
- 分部拆分到位，为阶段 2 的模型结构铺好了
- 行业规模、市占率有权威来源和年份
- O/V/T 三层已分开：O 池有直接竞争关系，V 池逐家说明估值可比理由，T 池逐家标产业链关系，目标公司已纳入并高亮
- staging 已覆盖三种横向视图所需的市值、收入、利润、PE、PS、增长及行业关键指标；期间、币种、行情日和 `N/M` 口径统一，阶段 3 可直接合并进正式 Excel
- 横向数据先写 staging 并校验，研究稿引用已验证快照，没有直接覆盖原始数据
- 表后给出定价差异与产业链利润池判断，不只复述排名
- 每个关键数有来源和日期，关键数据双源
- 全文按 `研报写作规范.md` 写，无 AI 味
