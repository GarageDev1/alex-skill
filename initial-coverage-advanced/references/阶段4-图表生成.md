## 图表数据：只画真值，缺数不补造

图表数据点和正文数字一样，受同一套取数纪律约束——不存在"图表是次要数据可以放水"。这是上一次出事最集中的地方（编造指数线、给基石/解禁/融资图凑假明细、股价图插值补点），这一阶段最该守住。

**每张图建 `_data` 底稿。** 每张图配一份数据底稿，列清每个数据系列的每个点来自哪——`系列 | 数值 | 口径 | 时间点 | 来源`。**底稿里指不到来源的系列，不画**。一张图只有总量真值、没有明细真值，就只画总量，别把它拆成编造的明细图。

**禁止合成、插值、示意填充。** 缺数就留空、标「未获取」，或换一种只用真值的画法，绝不 `np.interp` 补点、`np.random` 加噪声、或贴个"示意/近似"就编一条曲线。"示意/近似/约"只能标真实数据的精度，不能给不存在的数据当免责声明——一个标了"示意"的编造数字，和没标的，对读者危害一样。

**必备图缺数就降级或省略。** 某张必备图的数据不够，降级成只画已知值，或省略并注明"数据未获取"，不补造。少一张真图，好过多一张假图。

**出图后必跑反编造 lint。** 全部图生成后，跑 `scripts/anti_fabrication_lint.py`，把图表构建脚本和各 `_data` 底稿传进去，命中逐条清掉（指到真实来源或删系列）才进阶段 5。脚本只兜机械可检的痕迹（插值、噪声、示意标签、无源系列），看起来合理的错数还得靠异源验证。

## 图表规范

每张图：

- **标题**只写「[描述] [单位]」，图内不预写最终“图表 N”。阶段 5 在原生 Word 表格和图片全部排定后统一连续编号
- **来源**小字脚注「来源：公司公告、WIND、XX研究所」
- 300 DPI（`dpi=300`），尺寸适配 A4 单栏或双栏
- 文件名使用稳定资产 ID + 描述，如 `peer_v_growth_valuation.png`、`chart_segment_revenue.png`；图表清单另列建议插入顺序。不要把阶段 4 的临时顺序当最终编号
- 数据标签清晰，不重叠、不超出边界、不裁切；坐标轴有单位

### 坐标轴与边框

直角坐标图（柱状图、折线图、面积图、散点图）统一去掉上、右边框线，保留 X / Y 实线；所有图一律不画网格线：

```python
ax.spines['top'].set_visible(False)
ax.spines['right'].set_visible(False)
ax.spines['left'].set_visible(True)     # X / Y 实线保留
ax.spines['bottom'].set_visible(True)
ax.grid(False)                          # 全部图去网格线
```

上边线和右边线永远不要显示。饼图、雷达图、热力图、足球场图、瀑布图没有常规直角坐标轴，按各自惯例处理，不强套这条。

### 双 Y 轴（股价走势对比图）

个股股价和指数点位量纲差几十倍，放同一条 Y 轴个股线会被压平。用 `twinx()` 双轴，两条线都画绝对值、**都用日频逐日收盘**：

```python
import pandas as pd
# 个股、指数都从结算日 K 原始序列读（已落 _data 底稿），不插值、不重采样补点
stock = pd.read_csv('[公司]_daily_close.csv')   # 逐日收盘，结算口径
index = pd.read_csv('[指数]_daily_close.csv')   # 同区间逐日收盘
fig, ax = plt.subplots()
ax.plot(stock.date, stock.close, color=PALETTE[0], label='个股股价（元）')   # 左轴：个股绝对股价
ax2 = ax.twinx()
ax2.plot(index.date, index.close, color=PALETTE[3], label='指数（点）')      # 右轴：指数绝对点位
ax.set_ylabel('股价（元）'); ax2.set_ylabel('指数（点）')
# 左右轴各自图例，或合并图例；ax2 同样去掉 top spine
ax2.spines['top'].set_visible(False)
```

两条线都画绝对值，不要归一化到 100。**数据必须是日频逐日真实收盘**——个股、指数、可比公司价格一律取日频结算序列，落 `_data` 底稿，按真实日频点画线。不要周频重采样、更不要插值补点（上次正是靠插值凑周度平滑曲线出的事）。

### 双图并列尺寸

成对/可对照的图（分部增速对比、可比公司倍数、三情景、毛利率 vs 净利率等）在阶段 5 会一行两图各占 50% 栏宽并列。这类图按窄版生成——长宽比偏 4:3、`figsize` 收窄、字号适当放大，避免缩到 50% 后文字发虚发小。其余单栏全宽图按常规尺寸。阶段 5 用无边框二列表格放置（见 `Word文档格式标准.md`）。

## 图表清单（按数据来源映射）

40–70 张，覆盖前三阶段 + 外部数据。下面是骨架，按公司和行业增减。

**必备图（不可缺）**：

- 股价走势对比（个股 vs 指数，近 12 个月，封面用）：**双 Y 轴**——左轴个股绝对股价、右轴指数绝对点位，两者都是绝对值，**严禁指数化 / 归一化到 100**；**日频逐日真实收盘**（约 250 个交易日点），不插值、不重采样
- 分部营业收入（堆叠柱或堆叠面积，历史 + 预测）
- 盈利预测（收入/归母净利/增速，历史 + 5 年预测）
- 估值：SOTP 拆解图（多业务）或可比公司倍数对比
- O 池经营对标图（市占率 / 毛利率 / 关键经营指标，按行业选）
- V 池增长—估值矩阵（盈利公司用利润/EPS 增速 × PE FY1；未盈利/利润失真公司另做收入增速 × PS FY1）
- T 池产业链定价光谱（按产业链环节比较市值、增长、利润率和 PE/PS）
- 估值足球场图（各方法估值区间 + 当前价 + 目标价）
- DCF 敏感性热力图（若做了 DCF）
