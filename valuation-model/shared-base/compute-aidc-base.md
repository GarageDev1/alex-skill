## ① 总盘子:全球 AI 数据中心专项 CapEx($B)
> 口径=**AI datacenter capex**(严格锁 AI 维度,非 hyperscaler 总额)。2021-2024 为 AI 起步期粗估(很小,AI 2023 才起量);2025A 起为 AI 规模化期(本模型重点)。

口径 = 美国 Top5(MSFT/GOOGL/AMZN/META/ORCL)+ 中国 BAT+字节。

| 年 | 总额 | 性质 | 来源 |
|---|---|---|---|
| 2021A | ~15 | 粗估 | AI 起步期(极小) |
| 2022A | ~30 | 粗估 | AI 起步期 |
| 2023A | ~70 | 粗估 | AI 2023 起量 |
| 2024A | ~200 | 粗估 | AI 规模化前夜 |
| 2025A | **488** | 实际 | CreditSights / 财报(AI 规模化期) |
| 2026E | 830 | 公司指引 | TrendForce 2026/5;美四大指引合计已 >$700B |
| 2027E | 1,050 | 预测 | Goldman "Tracking Trillions" |
| 2028E | 1,240 | 预测 | Goldman 基线 |
| 2029E | 1,380 | 预测 | Goldman 基线 |
| 2030E | 1,500 | 预测 | Goldman 基线 |
| 2031E | 1,600 | 预测 | Goldman 基线 |
| 2032–2036E | 1,680→2,042 | 长期外推 | 5% CAGR |

增速锚:2027 +27% / 2028 +18% / 2029 +11% / 2030 +9% / 2031 +7% / 2032+ +5%。

**两个 sanity check(基座自带)**
- **占 GDP**:2026E $830B ≈ 全球 GDP($120T)的 0.7%——量级合理,不是离谱外推。
- **下行形态类比**:1996–2010 美国宽带 capex($21B→2000-01 峰值 $76B→2003 回落 $49B→缓升)。AIDC 见顶后大概率走同款"冲顶—回落—再平台"曲线,用来 stress-test 长期假设,别线性外推到天上。

> ⚠ **已知口径分歧**:NVDA 模型活公式行 2025A 写的是 448,但带逐区域来源的明细表是 488(US$443B + 中国$45B)。本基座取 **488**(有来源),旧 448 视为笔误。这正是基座要消灭的那种"同一模型内部对不上"。

## ② 单位成本拆解:$51.6B / 1GW

来源:BofA / 硅谷101《AI最烧钱的战场:数据中心的真实账单》。

**spend tree(MECE,求和=总额,用来切公司 SAM + 跑护栏)**

```
总 51.6  (100%)
├─ IT设备 43.15 (83.6%)
│   ├─ 服务器 37.5 (72.7%)
│   │   ├─ GPU(含HBM) 26.25 (50.9%)   ← =服务器×0.7
│   │   └─ 服务器其他(CPU/通用DRAM/主板/存储控制) 11.25 (21.8%)
│   ├─ 网络 3.75 (7.3%)
│   │   ├─ 交换机+网卡DPU 1.5 (2.9%)
│   │   └─ 光模块/线缆/其他 2.25 (4.4%)
│   └─ 存储 1.9 (3.7%)
├─ 供电 2.7 (5.2%)   柴油0.8 / 开关0.615 / UPS0.985 / 其他配电0.3
├─ 冷却 1.475 (2.9%) 冷却塔0.09 / 冷水机组0.36 / CDU0.45 / CRAH0.575
└─ 工程建设 4.28 (8.3%)
```

**chip content view(跨切口,与 tree 重叠,标 subset)**——存储器/逻辑厂在这里找自己那刀:

| 含量 | $/GW | 占总 | 嵌在哪 |
|---|---|---|---|
| HBM | 10.5 | 20.4% | ⊂ GPU(含HBM)(封装内,占服务器 ~28%) |
| 通用 DRAM(DDR5) | 3.5 | 6.8% | ⊂ 服务器其他 |
| NAND | 2.84 | 5.5% | 存储$1.9B + 服务器内置盘 |

## ③ 公司怎么切(Layer 2 示例)

| 公司 | 切哪几刀 | SAM/GW | 备注 |
|---|---|---|---|
| **NVDA** | GPU(含HBM)+ 交换机/网卡 | ≈ 27.75 | 卖整颗 GPU,HBM 是转售进来的 |
| **Hynix** | HBM + 通用DRAM + NAND | ≈ 16.84 | 存储三件套 |
| **TSMC** | GPU 内的逻辑代工价值 | GPU 含量 × 代工占 ASP 比 | 在 GPU 节点内再切代工层 |

**切片 → 营收公式骨架**:
```
公司切片 TAM($) = AIDC CapEx 总额(①) × 该公司部件占总 capex 的 %(②)
公司营收       = AI直驱段用 AIDC CapEx × 收入强度;其余周期段单独拍量价
```

---

## ④ 用基座时的两条纪律

1. **护栏只对 spend tree(最终需求口径)成立**:同一个 tree 节点被几家声称,各家隐含 $/GW 之和 ≤ 节点 $/GW。
2. **营收会跨层 resale 叠加,不能直接相加对总额**:Hynix 的 HBM($10.5/GW)已经包在 NVDA 的 GPU($26.25/GW)收入里。算"产业链总营收"时若把两者相加 = 重复计算。各公司营收对的是**各自所在层**,不是同一个总盘子。
