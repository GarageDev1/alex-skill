# 物理锚:消费品牌(终端动销 + 提价 + 渗透)

> 适用:白酒/饮料/食品/化妆品等品牌消费品。范例:贵州茅台、农夫山泉。

## 锚是什么
**终端动销量(销量) × 出厂/终端价**。销量由 `渗透率 × 人均消费量 × 目标人群` 驱动;价由 `提价节奏 + 产品结构升级(mix)` 驱动。

## 链 build pattern(骨架)
```
① 物理锚:  销量 = 目标人群 × 渗透率 × 人均消费频次/量
           价   = 出厂价(× 提价) ;  结构升级 = 高端占比↑ 拉动均价
② TAM:     品类总容量(行业销量 × 行业均价, 分价格带)
③ 营收:    公司份额 × TAM ;或 自家销量 × 自家均价(分产品线)
④ 利润:    毛利(高端 mix↑→毛利↑)→ 销售费用(品牌投放)→ 净利 → 账面/FCF
```

## 数据从哪来 + 怎么估
- **销量/份额/价**:公司财报分产品线 + 行业协会(如中国酒业协会)/尼尔森动销 + 可追溯的经销商调研材料。
- **提价**:接历史提价周期 + 品牌定价权(高端品牌可持续提价,大众品受限)。
- **渗透/人均**:接可比国家/世代人均消费曲线,给天花板;别把短期囤货当真实动销。
- **估值**:稳态高 ROE 品牌用 DCF + P/E(给品牌溢价倍数,落历史带内)。

## 要点
- **区分动销 vs 渠道压货**:营收认终端动销,渠道库存变化要剔除,否则高估。
- 量价拆开:量(渗透天花板约束)和价(提价+结构)各一条线,不要并成一个增速。
