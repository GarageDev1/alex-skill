# 业务驱动量:消费电子(出货量 × ASP × 渗透/换机)

> 适用:手机/PC/可穿戴等硬件。范例:Apple、小米。

## 基准是什么
**出货量(units) × ASP**,出货量由 `安装基数 × 换机周期` 或 `目标人群 × 渗透率` 驱动。

## 计算方法(结构)
```
① 业务驱动量:  出货量 = 安装基数 / 换机周期(成熟)  或  目标人群 × 渗透率(成长)
           × ASP(分档/分区)
② TAM:     行业出货量 × 行业 ASP(分区域/价格带)
③ 营收:    公司份额 × TAM;或自家出货 × 自家 ASP + 服务/生态收入
④ 利润:    硬件毛利 + 服务毛利(mix)→ opex → 净利 → 账面/FCF
```

## 要点
- 渗透率/换机周期 是可证伪的业务驱动量,要接历史曲线 + 可比国家/品类类比。
- 服务收入(高毛利)单独一条线,不要并进硬件。

## 数据从哪来 + 怎么估
- **出货量/份额/ASP**:IDC / Canalys / Counterpoint 等可追溯行业资料 + 公司财报分区域/分品类;可选检索工具仅加速定位。
- **换机周期**:安装基数 ÷ 年出货反推(成熟市场 3–4 年);**渗透率**:接可比国家/品类历史 S 曲线,给天花板。
- **ASP 分档**:高/中/低端各一条,不要 blended;新机型提价单列。
- **服务收入**:单独一条线(高毛利,估值给更高倍数),不要并进硬件。
