# 业务驱动量:周期品(供需平衡表 → 价格)

> 适用:大宗/能源/化工/部分内存价格周期。范例:油、铜、乙烯。

## 基准是什么
**供需平衡表**:供给(产能 × 开工率)vs 需求(下游消费),缺口 → 价格。价格是被推导量,不是直接设置的。

## 计算方法(结构)
```
① 业务驱动量:  供给 = Σ 产能 × 开工率(逐产线/逐国)
           需求 = Σ 下游消费(分终端)
           缺口/库存天数 → 价格(弹性/历史回归)
② (无独立 TAM,价格即核心输出)
③ 营收:    公司产量 × 价格(- 成本曲线位置)
④ 利润:    价格 − 现金成本 → 毛利 → 净利;强周期,跨周期看 mid-cycle
```

## 要点
- 价格**必须从供需缺口推**,不许直接设置价格曲线。
- 估值用 mid-cycle EV/EBITDA 或 P/B,避免用峰值/谷值盈利。

## 数据从哪来 + 怎么估
- **供需平衡表**:供给逐产线/逐国(产能 × 开工率,接扩产计划)+ 需求分终端(下游增速)。来源:行业协会 / IEA·USGS / 卖方大宗团队公开或用户提供材料。
- **缺口 → 价格**:库存天数 or 缺口率对价格做历史回归/弹性,得价格,不许直接设置价格曲线。
- **成本曲线**:把公司放在行业现金成本分位(决定周期底部能否活)。
- **估值**:mid-cycle EV/EBITDA 或 P/B(用归一化盈利),避免峰值/谷值盈利。
